Risk-Adjusted Return – Sample Code

Example Python function for a simple risk-score calculation used in internal analytics: 


def risk_adjusted_return(expected_return, risk_free_rate, volatility):
    """
    Computes a simplified Sharpe-like ratio.

    expected_return : float
        Expected portfolio return (annualized, e.g. 0.12 for 12%)
    risk_free_rate : float
        Risk-free rate (e.g. 0.06 for 6%)
    volatility : float
        Standard deviation of returns (e.g. 0.18 for 18%)

    Returns
    -------
    float
        Risk-adjusted return measure.
    """
    if volatility == 0:
        return 0.0
    return (expected_return - risk_free_rate) / volatility

print(risk_adjusted_return(0.12, 0.06, 0.18))

Core Valuation Formulas (Math Support)

Present Value (PV) of a single future cash flow:

\[ PV = \frac{CF}{(1 + r)^n} \]

Net Present Value (NPV) of a series of cash flows:

\[ NPV = \sum_{t=0}^{T} \frac{CF_t}{(1 + r)^t} \]

Basic compound interest formula:

\[ A = P (1 + r)^n \]
